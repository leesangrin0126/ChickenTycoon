# ChickenTycoon

ChickenTycoon is a chicken-themed endless tycoon game built with Python and Pyxel.

## 🎮 Overview

This project is an endless growth game where the player develops a chicken business through repeated earning, upgrading, and progression.

The game does not have a fixed ending, so players can continue growing their business without a final limit.

## ✨ Features

- Chicken-themed tycoon gameplay
- Endless growth system
- Click-based earning system
- Upgrade and progression system
- Pixel-style game screen
- Simple interactive UI
- Game assets managed with a Pyxel resource file
- Return to the first menu by pressing `Q`

## 🛠 Tech Stack

- Python
- Pyxel

## 🚀 How to Run

Pyxel is required to run this game.

1. Install Pyxel.

```bash
pip install pyxel
```

2. Run the game.

```bash
python app.py
```

## 🎮 Controls

- Use the mouse to interact with the game.
- Press `Q` to return to the first menu.
